# Vanilla Sidebar – Two-step Expand

This project is a modern, animated sidebar UI built with HTML, CSS, and JavaScript. It features a two-step expand/collapse interaction, dynamic content panels, and a clean, responsive design inspired by professional dashboard layouts.

## Features
- **Two-step sidebar:**
  - Icon-only panel for quick navigation
  - Expandable panel with full text and content
- **Dynamic content:**
  - Dashboard, Tasks, and Mail sections, each with their own menu and submenus
  - Content updates instantly when switching sections
- **Modern UI/UX:**
  - FontAwesome icons throughout
  - Animated transitions for expanding/collapsing
  - Highlighted active section
  - Common search bar in all expanded panels
  - Responsive and theme-aware (light/dark)
- **Customizable:**
  - Easily add or modify sidebar sections and content in `script.js`

## Files
- `sidebar.html` – Main HTML structure
- `style.css` – All styles and responsive design
- `script.js` – Sidebar logic, dynamic content, and animation

## Usage
1. Open `sidebar.html` in your browser.
2. Click any main icon to switch sections.
3. Click the expand arrow to open the full sidebar; click the collapse arrow to return to icon-only mode.
4. Use the search bar in any expanded panel.
5. Toggle light/dark mode with the theme icon in the sidebar.

## Customization
- To add or change sidebar sections, edit the `panelTemplates` object in `script.js`.
- To change icons, use any [FontAwesome 6 Free](https://fontawesome.com/icons) icon class.
- To adjust colors or spacing, edit variables and rules in `style.css`.
